# Phase 4 Implementation Plan：私聊 / poke 指令面

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把指令处理扩展到私聊文本与 poke 入口；回复回到来源通道（频道→频道，私聊/poke→私信）。

**Architecture:** 三入口都走 `Event::Message`（按 `target` 区分）。`ChatMessage` 带 `reply_to`，回复管道由 `String` 升级为 `Reply{dest,text}`，driver 按 dest 路由（频道文本 / 私信文本）。权限按 uid 跨入口统一，不变。

**Tech Stack:** 现有 workspace；tsclientlib `MessageTarget` + c2s 文本（target=Client）。

**前置:** Phase 3 已合并（37 测试）。bin **无需改动**（reply 通道类型由推断自动从 String 变 Reply）。

/// 回复目标。
#[derive(Clone, Copy)]
pub enum ReplyDest {
    Channel,
    Client(ClientId),
}

/// 一条待发回复。
pub struct Reply {
    pub dest: ReplyDest,
    pub text: String,
}

/// 从入口 target 计算回复目标；Server 广播返回 None（不处理）。
pub fn reply_dest_for(target: &MessageTarget, invoker_id: ClientId) -> Option<ReplyDest> {
    match target {
        MessageTarget::Channel => Some(ReplyDest::Channel),
        MessageTarget::Client(_) => Some(ReplyDest::Client(invoker_id)),
        MessageTarget::Poke(_) => Some(ReplyDest::Client(invoker_id)),
        MessageTarget::Server => None,
    }
}
```

/// 向某客户端发送私信文本。
fn send_private_text(con: &mut Connection, client_id: ClientId, text: &str) -> Result<()> {
    let cmd = c2s::OutSendTextMessageMessage::new(&mut std::iter::once(c2s::OutSendTextMessagePart {
        target: TextMessageTargetMode::Client,
        target_client_id: Some(client_id),
        message: text.into(),

    Ok(())
}

- [ ] **Step 7: commands —— 用 Reply 回复**

在 `crates/commands/src/lib.rs`：把 `use ts_connection::ChatMessage;` 改为：

use ts_connection::{ChatMessage, Reply, ReplyDest};
```
把 `run` 改为（`reply_tx: Sender<Reply>`，每次回复带 `dest: msg.reply_to`）：
```rust
pub async fn run(
    mut chat_rx: mpsc::Receiver<ChatMessage>,
    handle: PlayerHandle,
    snapshot: Arc<Mutex<Snapshot>>,
    store: Store,
    perms: Permissions,
    reply_tx: mpsc::Sender<Reply>,
) {
    while let Some(msg) = chat_rx.recv().await {
        let Some(cmd) = parse(&msg.text) else { continue };
        let name = command_name(&cmd);
        if name != "help" && name != "whoami" && !perms.allows(&msg.invoker_uid, name) {
            let text = format!("⛔ 无权限：{name}（你的角色：{}）", perms.role_of(&msg.invoker_uid));
            let _ = reply_tx.send(Reply { dest: msg.reply_to, text }).await;
            continue;
        }
        let text = handle_command(cmd, &handle, &snapshot, &store, &msg.invoker_uid).await;
        let _ = reply_tx.send(Reply { dest: msg.reply_to, text }).await;
    }
}
```
（`handle_command`、`command_name`、解析、权限逻辑不变。`msg.reply_to` 是 `Copy`，用两次没问题。）
